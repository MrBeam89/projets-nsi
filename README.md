# projets-nsi
Projets en classe de NSI
## NCI (Numerical Converting Interface)
Outil pour convertir des nombres en bases différentes
- Base 10 (Décimal)
- Base 2 (Binaire)
- Base 16 (Hexadécimal)

Dispose d'une version en ligne de commande et d'une avec une interface graphique
